## 🖥️ Infrastructure Management System (OOP)

This project demonstrates the power of **Object-Oriented Programming** applied to DevOps and Backend infrastructure. 

### 🚀 Key Features
* **Class Inheritance:** Specialized `DatabaseServer` and `WebServer` inheriting from a base `Server` class.
* **Polymorphism:** Managing a fleet of different server types using a single automated loop.
* **Visual Feedback:** Custom ASCII progress bar to simulate system boot/reboot sequences.
* **Validation Logic:** Integrated checks to prevent operations (like queries) on offline resources.

### 🛠️ Tech Stack
* **Language:** Python 3.x
* **Core Concepts:** Encapsulation, Inheritance, Type Hinting, and `__name__ == "__main__"` guards.
